# 🚀 Tamil News Website - Quick Start

## ⚡ Fastest Way to Get Started

### 1. **Run Setup Script**
```bash
setup.bat
```

### 2. **Start Development Servers**
```bash
start-dev.bat
```

### 3. **Access Your Applications**
- **Main Website**: http://localhost:3000
- **Strapi Admin**: http://localhost:1337/admin

## 🔧 Essential Configuration

### **Supabase Setup** (Required)
1. Go to https://supabase.com and create a project
2. Copy your project URL and anon key
3. Update `.env.local` with your credentials
4. Run the SQL script in Supabase SQL Editor:
   ```sql
   -- Copy content from scripts/001_create_profiles_and_subscriptions.sql
   ```

### **Strapi First-Time Setup**
1. Open http://localhost:1337/admin
2. Create your admin account
3. Content types are already configured!
